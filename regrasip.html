Regras para telefones IP



Ao contratar um telefone, o cliente assume uma fidelidade de 12 meses.
Prazo para envio: 45 dias
Os envios ocorrem sempre as terças e quintas
Código de rastreio é/será enviado para o email de cadastro
Cliente fica responsável por autenticar o MAC no painel O telefone é intransferível
DEVOLUÇÃO
O cliente deve devolver o telefone, antes de solicitar o cancelamento.
O cancelamento precisa ser autorizado após verificação do estado do telefone.
Caso o telefone tenha alguma avaria, o cliente deve arcar com o prejuízo
O cliente pode optar por comprar o equipamento pagando o saldo residual.
Caso o cliente se negue a devolver o equipamento será acionado judicialmente.
DEFEITOS e PROBLEMAS
O cliente é responsável pelo envio do equipamento. O mesmo deve estar devidamente embalado, e contendo todas as peças (anexo). Caso esteja faltando alguma peça, o cliente fica responsável por arcar com os custos conforme tabela:
fonte: R$ 30,00
Base: R$ 50
Telefone: 279,90
Monofone: R$ 100,00
Os custos de reenvio nesses casos é da falepaco.
PROBLEMAS COM A ENTREGA
O envio dos equipamentos ocorre por correio É de responsabilidade do cliente confirmar as informações do endereço (não nos responsabilizamos por dados de endereço errado).
Caso o equipamento retorne para a Falepaco (não atendeu, endereço errado/inexistente etc..) o cliente deve pagar o frete novamente, ou optar pelo valor referente à instalação (não há devolução do valor de frete nesses casos).
